# 📝 To-Do List App (Full Stack)

## 🚀 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- Database: MySQL

## ⚡ Features
- Add new tasks
- View task list
- Delete tasks

## ▶️ How to Run
1. Clone repo
2. Setup database:
   ```sql
   CREATE DATABASE todo_db;
   USE todo_db;
   CREATE TABLE tasks (id INT AUTO_INCREMENT PRIMARY KEY, name VARCHAR(255));
